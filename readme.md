#stamp

save templates from directories.

```
usage: stamp [-h] [-l] [-s <name>] [-p <name>] [-P <name>] [-x <name>]

create and save templates from directories

optional arguments:
  -h, --help  show this help message and exit
  -l          list all saved templates
  -s <name>   create a template from the current directory as <name>
  -p <name>   paste template <name> into the current directory
  -P <name>   same as -p, but overwriting existing files
  -x <name>   delete a template by name
```
